# Размещение файла архитектуры

## Решение

Перенести единственную актуальную схему `Архитектура LLM-бота Мороз и Солнце.excalidraw` из корня проекта в `project/data/architecture/` без изменения содержимого.

## Обоснование

- корень проекта остаётся чистым;
- схема отделена от исходных клиентских материалов в `project/data/source_documents/`;
- назначение папки понятно без дополнительных правил.

## Сопутствующие изменения

Обновить все ссылки на схему в `AGENTS.md`, `Дорожная карта.md` и других отслеживаемых документах. Записать перенос в `changelog.md` и проверить, что старого пути больше нет, новый файл является валидным Excalidraw JSON, а Git видит перенос.

## Границы

Не менять содержимое схемы, не перемещать ТЗ и не перестраивать остальные папки `project/data`.
