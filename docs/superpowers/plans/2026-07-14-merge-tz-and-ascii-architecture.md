# Merge TZ and ASCII Architecture Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Объединить актуальное ТЗ и ASCII-архитектуру в единый главный документ `ТЗ и архитектура.md` без потери требований и архитектурных связей.

**Architecture:** Существующее ТЗ остаётся основой документа. Его раздел 16 заменяется полной ASCII-архитектурой, после чего отдельный Markdown-файл схемы удаляется, а Excalidraw остаётся визуальным исходником.

**Tech Stack:** Markdown, Excalidraw JSON, Git, PowerShell/Python для проверок.

## Global Constraints

- Не менять смысл требований исходного ТЗ.
- Не менять содержимое Excalidraw.
- Сохранить все 66 подписанных блоков архитектуры.
- Не переписывать исторические записи changelog.
- Не отправлять изменения на GitHub.
